NAME
	isascii – test for ASCII character

LIBRARY
	Standard C Library (libc, -lc)

SYNOPSIS
	#include <ctype.h>
	int	isascii(int c);

DESCRIPTION
	The function tests for an ASCII character, which is any character between 0 and octal 0177 inclusive.
	octal 0177 = decimal 127

SEE ALSO
	ctype(3), iswascii(3), ascii(7)
