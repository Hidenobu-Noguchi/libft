about memset

#include <string.h>
void	*memset(void *s, int c, size_t n);
Returns a pointer to the memory area [s].

DESCRIPTION
The memset() function writes [n] bytes of value [c](converted to an unsigned char) to the string [s].
fill the first [n] bytes of the memory area pointed to by [s] with the constant byte [c].

RESULT
memset(str0 + 10, 'a', 10 * sizeof(char))
str0[30] = 1234567890aaaaaaaaaa123456789

memset(str1, 97, 10 * sizeof(char))
str1[30] = aaaaaaaaaa1234567890123456789
