NAME
	memchr – locate byte in byte string

LIBRARY
	Standard C Library (libc, -lc)

SYNOPSIS
	#include <string.h>
	void	*memchr (const void *s, int c, size_t n);

DESCRIPTION
	The function locates the first occurrence of [c](converted to an unsigned char) in string [s].

RETURNS
	The function returns a pointer to the byte located, or NULL if no such byte exists within [n] bytes.

SEE ALSO
	strchr(3), strcspn(3), strpbrk(3), strrchr(3), strsep(3), strspn(3), strstr(3), strtok(3), wmemchr(3)
